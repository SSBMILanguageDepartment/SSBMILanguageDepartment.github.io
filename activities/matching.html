Esak’ahá daak’ábe mi c’aykɨ́? (Do you want to know more?)

The "Animals" matching games include Nisenan and Miwok animal names that we use when we play "Welk’óy!"/"Wélse’!" (Go Fish!).

The matching games were created using a LaTeX computer script that positions a set of words and images on a page. By using computational tools to automate creating matching games, we can quickly create new matching activities using different sets of Nisenan and Miwok words.

If you would like a copy of the matching games and cannot print them at home, please visit the SSBMI Language Department in the SSBMI Tribal Administration building or contact us at language@ssband.org.
