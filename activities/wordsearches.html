Esak’ahá daak’ábe mi c’aykɨ́? (Do you want to know more?)

The "Animals" word searches include Nisenan and Miwok animal names that we use when we play "Welk’óy!"/"Wélse’!" (Go Fish!).

The word searches were created in part using a Python computer script that randomly positions a set of words within a grid and then randomly generates Nisenan or Miwok letters to fill the rest of the grid. By using computational tools to automate creating word searches, we can quickly create new word search activities using different sets of Nisenan and Miwok words.

If you would like a copy of the word searches and cannot print them at home, please visit the SSBMI Language Department in the SSBMI Tribal Administration building or contact us at language@ssband.org.
